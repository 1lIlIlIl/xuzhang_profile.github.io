# Xu Zhang — Personal Research & Portfolio Website

A minimalist, architecture-inspired personal website showcasing research in flexible electronics and neuromorphic computing. Designed with influences from Virgil Abloh, Matthew Williams, and Maison Margiela.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Interactions**: Subtle animations and transitions for enhanced UX
- **Modern Typography**: Playfair Display for headings, Montserrat for body text
- **Accessible Navigation**: Fixed header with smooth scroll navigation
- **Performance Optimized**: Minimal external dependencies, optimized animations

## Project Structure

## Customization

### Adding Your GitHub/LinkedIn
Edit the contact links in `index.html`:
```html
<a href="https://github.com/yourusername" class="contact-link" target="_blank">
```

### Modifying Colors
Update CSS variables in `css/style.css`:
```css
:root {
    --color-primary: #000000;
    --color-secondary: #ffffff;
    /* ... */
}
```

### Editing Project Details
Update project information in the Projects section of `index.html`.

## Deployment

### GitHub Pages
1. Create a GitHub repository named `yourusername.github.io`
2. Push this project to that repository
3. Your site will be available at `https://yourusername.github.io`
